# リソース

- [Exercism](https://exercism.org/) - 70以上の言語で無料のコーディング演習とメンタリング。実際の課題で基礎を練習。
- [W3Schools](https://www.w3schools.com/) - HTML、CSS、JavaScriptなどのチュートリアルとリファレンス。素早い検索とインタラクティブな例に最適。
- [MDN Web Docs](https://developer.mozilla.org/en-US/) - Web技術の決定版リファレンス。HTML、CSS、JavaScript、Web APIの詳細なドキュメント。
- [Hello Interview](https://www.hellointerview.com/) - システム設計とコーディング面接の準備。実際のシステムがどう構築・評価されるかを学ぶ。
- [React](https://react.dev/) - React公式ドキュメント。Reactを作るチームからコンポーネント、フック、モダンなパターンを学ぶ。
- [TypeScript](https://www.typescriptlang.org/docs/) - TypeScript公式ハンドブック。JavaScriptに型安全性を追加し、バグをリリース前に捕まえる。
- [Nest.js](https://nestjs.com/) - スケーラブルなサーバーサイドアプリのためのNode.jsフレームワーク。モジュール、デコレーター、依存性注入を適切に実装。
