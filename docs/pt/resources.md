# Recursos

- [Exercism](https://exercism.org/) - Exercícios gratuitos de programação em mais de 70 linguagens com mentoria. Pratique os fundamentos com desafios reais.
- [W3Schools](https://www.w3schools.com/) - Tutoriais e referências para HTML, CSS, JavaScript e mais. Ótimo para consultas rápidas e exemplos interativos.
- [MDN Web Docs](https://developer.mozilla.org/en-US/) - A referência definitiva para tecnologias web. Documentação profunda de HTML, CSS, JavaScript e Web APIs.
- [Hello Interview](https://www.hellointerview.com/) - Preparação para entrevistas de design de sistemas e programação. Aprenda como sistemas do mundo real são construídos e avaliados.
- [React](https://react.dev/) - A documentação oficial do React. Aprenda componentes, hooks e padrões modernos direto da equipe que constrói o React.
- [TypeScript](https://www.typescriptlang.org/docs/) - O manual oficial do TypeScript. Adicione segurança de tipos ao JavaScript e capture bugs antes que cheguem à produção.
- [Nest.js](https://nestjs.com/) - Framework Node.js progressivo para aplicações server-side escaláveis. Módulos, decoradores e injeção de dependência bem feitos.
