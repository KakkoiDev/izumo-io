# R17: 英語の重要性

英語はテクノロジー世界の共通語です。完璧である必要はありませんが、実用的な習熟は開発者として身につけられる最も効果の高いスキルの一つです。ドキュメント、チュートリアル、フォーラム、求人、オープンソースはどれもデフォルトで英語です。
{: .lesson-intro }

## なぜ重要か

プログラミング言語自体が英語です: `function`、`return`、`class`、`import`。エラーメッセージも英語。React、Node.js、Python、そしてほぼあらゆる主要ツールの公式ドキュメントは、まず英語で書かれます。翻訳は数週間から数ヶ月遅れ、そもそも出ないこともある。英語なしでは、常に誰かが翻訳するのを待つ側です。

国際チームは英語でコミュニケーションします。リモート職はしばしば英語必須。コードレビュー、PR説明、コミットメッセージ、仕様 - ほとんどの企業で英語。技術的アイデアを英語で明確に表現できることは、技術スキルだけでは開かない扉を開きます。

## 上達の方法

- 翻訳版ではなく英語でドキュメントを読む
- 英語の技術トークを見る(字幕OK)
- コミットメッセージ、コメント、READMEを英語で書く
- 英語圏コミュニティに参加する(GitHub、Discord、フォーラム)
- 完璧ではなく明確なコミュニケーションを目指す
