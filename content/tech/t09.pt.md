# T09: CSS no Limite

CSS pode fazer muito mais do que mudar cores. Com transições, animações e seletores espertos, dá para construir componentes interativos sem uma única linha de JavaScript. É como descobrir que o seu pincel também esculpe.
{: .lesson-intro }

## Transições

Transições animam mudanças de propriedade de forma suave. Especifique qual propriedade animar, quanto tempo leva e a função de easing.

```
.button {
    background: #3498db;
    transition: background 0.3s ease, transform 0.2s ease;
}
.button:hover {
    background: #2980b9;
    transform: scale(1.05);
}
```

## Animações por Keyframes

Para movimentos mais complexos, defina keyframes com estados específicos em diferentes pontos no tempo.

```
@keyframes fadeIn {
    from { opacity: 0; transform: translateY(-10px); }
    to { opacity: 1; transform: translateY(0); }
}
.card { animation: fadeIn 0.5s ease-out; }
```

## Accordion Puro em CSS

O pseudo-classe `:target` permite alternar visibilidade baseado no hash da URL, criando componentes interativos sem JavaScript.

```
.panel { max-height: 0; overflow: hidden; transition: max-height 0.3s; }
.panel:target { max-height: 500px; }
```

<li>A pseudo-classe :target habilita padrões interativos sem JS</li>
<li>Sempre considere performance - anime transform e opacity, não width ou height</li>
