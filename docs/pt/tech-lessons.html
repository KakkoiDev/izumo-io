Aulas Técnicas

Aulas práticas de HTML a aplicações de IA full stack. Construa projetos reais passo a passo.

Fase 1: A Bancada (Configuração do Ambiente)

Instale as ferramentas antes do primeiro corte

Um artesão prepara a bancada antes de começar a primeira peça

T01

Configuração do Ambiente

Instale o VS Code, o Node.js, o Git e um navegador. Verifique pelo terminal. Escreva hello.html e veja rodar.

Fase 2: A Linha de Comando (Terminal)

Linha direta com a máquina - texto vence cliques em escala

Botões são para turistas. Comandos são para residentes

T02

Terminal do Zero

Navegar, criar, apagar arquivos. Ler o manual. Aprender o PATH e truques de velocidade.

 T03

Pipes e Ferramentas de Poder

stdin/stdout/stderr, pipes, grep/find/sed/awk, encadeamento, pequenos scripts de shell.

Fase 3: O Livro (HTML/CSS)

Estrutura e decoração, nenhum JavaScript

Como construir um livro: esqueleto (HTML) e design (CSS)

T04

Hello World

Sua primeira página web. Crie um arquivo, abra num navegador.

 T05

Tags HTML

Os blocos de construção: cabeçalhos, parágrafos, listas, links, imagens.

 T06

Formulários HTML e Blocos

HTML interativo: input, select, textarea, validação, details/summary.

 T07

Fundamentos de CSS

Deixando as coisas bonitas: seletores, cores, fontes, box model.

 T08

Layout com CSS

Organizando a página: flexbox, grid, design responsivo, media queries.

 T09

CSS no Limite

Interatividade pura em CSS: animações, transições, :target, accordion/tabs.

Fase 4: O Cérebro (JavaScript)

Adicionando pensamento e interação à página

Como adicionar músculos e um cérebro ao seu livro

T10

Introdução ao JavaScript

Primeiros passos com JS: console.log, variáveis, tipos, funções.

 T11

Manipulação do DOM

Controlando a página: querySelector, createElement, eventos.

 T12

Formulários e Dialog

Lidando com entrada do usuário: validação de form em JS, elemento dialog.

 T13

Estruturas de Dados

Organizando informação: arrays, objetos, loops, listas dinâmicas.

 T14

Persistência

Lembrando dados: localStorage, JSON parse/stringify.

 T15

Fetch API

Falando com servidores: requisições HTTP, async/await, promises.

 T16

Site Dinâmico: Roteamento

Uma página, várias visualizações: hash routing, componentes, estado.

 T17

Site Dinâmico: Offline

Funciona sem internet: service workers, PWA, cache.

 T18

Site Dinâmico: Acabamento

Qualidade de produção: performance, lazy loading, tratamento de erros.

Fase 5: A Máquina do Tempo (Git e GitHub)

Rastreie cada mudança, colabore sem pisar no pé um do outro

Git é sua máquina do tempo. GitHub é a oficina compartilhada onde os viajantes do tempo se encontram

T19

Fundamentos de Git

A máquina do tempo: três áreas, commits, branches, desfazer sem medo.

 T20

GitHub e Colaboração

A oficina compartilhada: remotes, push/pull, GitHub Flow, pull requests.

Fase 6: A Cozinha (Backend)

Servidores, APIs e bancos de dados

Construindo a cozinha do restaurante que prepara a comida

T21

Servidor Node.js

Construindo a cozinha: servidor HTTP, servindo arquivos, tratando requisições.

 T22

Endpoints de API

O cardápio: design REST, respostas JSON, roteamento.

 T23

Banco JSON

O livro de receitas: ler/escrever arquivos, CRUD com JSON.

 T24

SQLite

O arquivo suspenso: SQL, tabelas, CRUD, schema, chaves estrangeiras.

 T25

Autenticação

A conferência de identidade: sessões, cookies, hashing de senha, login/logout.

Fase 7: O Chef Robô (IA)

Integrando inteligência artificial

Contratando um robô que leu milhões de livros para ajudar na sua cozinha

T26

Ollama e Chat

O chef robô: setup de LLM local, API de chat, interface de chat.

 T27

App Full Stack com IA

O restaurante inteiro: conectar frontend, backend, IA e banco.

Fase 8: O Kit de Franquia (Frameworks Frontend)

UI baseada em componentes e JavaScript com tipagem segura

Atualizando de um cardápio desenhado à mão para um sistema PDV profissional

T28

Fundamentos de React

Pensando em componentes: JSX, props, tratamento de eventos, renderização de listas.

 T29

Estado e Efeitos no React

UIs interativas: useState para memória, useEffect para efeitos colaterais.

 T30

TypeScript

Segurança de tipo: anotações, interfaces, tipando componentes React.

 T31

Next.js: Roteamento e Renderização

Roteamento baseado em arquivos, server vs client components, layouts.

 T32

Next.js: Dados e API

Busca de dados em server components, rotas de API, montando uma página completa.

Fase 9: A Sede (Frameworks Backend)

Arquitetura backend de nível corporativo com Nest.js

Construindo a cozinha corporativa que gerencia todas as filiais da franquia

T33

Nest.js: Arquitetura

Módulos, controllers, services e injeção de dependência.

 T34

Nest.js: Dados e Autenticação

DTOs, validação, integração com banco, guards de autenticação.

Fase 10: O Kit de Lego (Web Components)

Invente suas próprias tags HTML - componentes nativos da plataforma que funcionam em qualquer framework

Peças de Lego embutidas no navegador. Encaixe-as, sem precisar de framework

T35

Web Components I

Invente suas próprias tags: Custom Elements, Shadow DOM, :host e ::part.

 T36

Web Components II

Templates, slots, callbacks de ciclo de vida e o Lit como camada de açúcar.

Fase 11: A Planta (Design de Sistemas)

Arquitete sistemas grandes: escala, armazenamento, cache, trade-offs

Arquitetos desenham plantas antes que alguém derrame concreto

T37

Design de Sistemas: O Framework

Requisitos, entidades, API, design de alto nível, deep dives, trade-offs.

 T38

Design de Sistemas: Escala e Armazenamento

Vertical vs horizontal, SQL vs NoSQL, replicação, sharding, teorema CAP.

 T39

Design de Sistemas: Caching, Filas e Padrões

Cache-aside, filas, load balancers, CDNs, monolito vs microservices, números para decorar.
